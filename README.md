# Customer-Retention-Churn-Analysis
Customer Retention & Churn Analysis

This project analyzes customer churn behavior in a telecom business using Excel, SQL, Python, and Power BI. The objective was to understand customer retention patterns, identify churn drivers, and evaluate the financial impact of customer attrition.

The project began with data preparation and validation in Excel, followed by data cleaning and analysis using SQL. Exploratory Data Analysis (EDA) was performed in Python to uncover trends, patterns, and customer behavior insights. Finally, Power BI was used to create interactive dashboards that provide a comprehensive view of customer churn and revenue impact.

Key areas covered:

- Customer retention and churn overview
- Customer demographics and service usage analysis
- Churn driver identification
- Revenue loss and retention analysis
- Contract, payment method, and tenure-based insights
- Business recommendations for improving retention

Tools Used:

- Excel
- SQL
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- DAX

This project demonstrates an end-to-end analytics workflow, from data preparation and exploration to visualization and business storytelling.
